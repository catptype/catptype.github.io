Showcase

Real‑time Vehicle Detection & License Plate Recognition System

I developed the core AI engine and data collection system that evolved from an embedded prototype into a central component of a commercial security product. The system performs real-time vehicle analysis and Thai license plate recognition from live CCTV feeds.

Key Features & Architecture:

  • Data Collection & Labeling Tool: The system has a Dataset Mode to help create better training data. In this mode, the system automatically saves all prediction results to a database. Then, the built‑in web UI can be used to review, correct, and label the saved data. This process creates a clean, high quality dataset for future model training.
  • High-Performance Multi-Threading:
    1. Video Capture (RTSP)
    2. AI Inference & Processing
    3. Real‑time Result Broadcasting (WebSockets)
    4. System Health Monitoring (WebSockets)
  • Deployment Journey: The initial prototype was deployed on a Raspberry Pi (using NCNN for CPU optimization), while the production version is containerized with Docker for Linux servers.

Commercial Integration:

  • This project enables key features in the Venus Sentinel, such as automated gate barrier access for registered vehicles.
  • My application was architected for a dual role:
    • It provides a staff‑only web UI for system configuration.
    • Its backend was expanded to communicate detection data to the central platform via a REST API.
  • A collaborative effort, developed in cooperation with team members from different company departments.
