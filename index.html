YOLO Inference Toolkit

I developed this high-level Python toolkit to simplify and structure the process of running inference with Ultralytics YOLO models. By applying an object-oriented design, the toolkit abstracts away repetitive boilerplate code, allowing developers to focus on integrating AI into their applications rather than on the complexities of the underlying framework.

Core Design & Workflow:

  • Object-Oriented Architecture: The toolkit is built around a parent YoloModel class, which provides core functionalities. Specialized child classes inherit from it for each specific task:
    • YoloDetection for object detection
    • YoloClassification for image classification
    • And others for Segmentation and Pose Estimation...
  • Simplified "Load → Predict → Decode" Flow: The entire inference process—for both single images and batches—is streamlined into three intuitive steps. This pattern transforms raw model outputs into clean, human-readable Python data structures that are ready for immediate use.

Example: Simplified Classification

# 1. Initialize the model
yolo_model = YoloClassification("yolov8n-cls.pt")

# 2. Predict the image
raw_results = yolo_model.predict(image)

# 3. Decode for a clean output
class_name, score = yolo_model.decode_top1(raw_results)

print(f"Class: {class_name}")
# > "goldfish"

Example: Simplified Detection

# 1. Initialize the model
yolo_model = YoloDetection("yolov8n.pt")

# 2. Predict the image
raw_results = yolo_model.predict(image)

# 3. Decode for clean results
decoded_results = yolo_model.decode_results(raw_results)

# 4. Process the simple list
for box, score, name in decoded_results:
print(f"Found: {name}")
